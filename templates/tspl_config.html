Help
Setup Steps:
  1. Connect your TSPL barcode printer
  2. Note down the IP address and port
  3. Configure host and port below
  4. Click "Test Connection"
  5. Click "Test Print" to verify
  6. Enable and save configuration
Printer Info:
  • Default Port: 9100
  • Protocol: TSPL (Thermal)
  • Format: 100mm x 150mm labels
  • Supports: UTF-8 (Turkish chars)
Troubleshooting:
  • USB: Use localhost
  • Network: Use printer IP
  • Windows: Install manufacturer driver
  • Linux: Check /dev/usb/*
Note: PNG files are always generated. TSPL is optional.
Usage

When generating QR codes:

  1. Go to Parts → Generate QR
  2. Enter quantity
  3. Check "Print to TSPL" (if enabled)
  4. Click Generate
  5. Labels will print automatically
