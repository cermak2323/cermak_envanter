PROBLEM: "Flask backend failed to start"
SOLUTION:
  1. Check Python is installed: python --version
  2. Check requirements: pip install -r requirements.txt
  3. Test Flask directly: python app.py
  4. Check port 5002 is free: netstat -ano | findstr :5002

PROBLEM: "Electron window won't open"
SOLUTION:
  1. Wait 30 seconds (Flask might still starting)
  2. Check Flask console logs
  3. Verify Flask health: curl http://localhost:5002/health
  4. Check app.py for errors

PROBLEM: "npm not found"
SOLUTION:
  1. Install Node.js from https://nodejs.org
  2. Add Node.js to PATH environment variable
  3. Restart terminal/IDE
  4. Verify: npm --version

PROBLEM: "PyInstaller not found"
SOLUTION:
  1. Install it: pip install PyInstaller
  2. Verify: pyinstaller --version
  3. Run build script again

PROBLEM: "Port 5002 already in use"
SOLUTION:
  1. Kill existing process: taskkill /F /IM python.exe
  2. Or change port in app.py
  3. Run again
